# CF-dns v1.1.0

## 新版说明

本次版本重点修复了 DNS 记录分页、手机端布局和 Cloudflare API 连通性问题，适合直接作为新版发布。

## 更新内容

### 1. DNS 记录分页
- DNS 记录接口改为分页返回。
- 前端新增上一页、下一页和页码输入，支持直接跳转到指定页。
- 分页区改为显示总页数和总记录数。

### 2. 手机端显示优化
- DNS 记录列表改为中间区域单独横向滑动，不再撑出整个页面。
- 修复手机端第 11 条及后续记录不显示的问题。
- 调整分页按钮、批量按钮、添加记录按钮和弹窗按钮尺寸与布局。
- 收紧“管理域名”和“批量修改”区域的多余留白。
- 加载状态改为显示在列表中央，并增加加载动画。

### 3. PC 端和交互优化
- 调整域名头部结构，优化添加记录和批量操作按钮排序。
- 修复分页区按钮布局与对齐问题。
- 移除部分多余图标，界面更统一。

### 4. Cloudflare API 连接修复
- 为 `api.cloudflare.com` 请求增加公共 DNS 解析兜底。
- 在本机 DNS 异常时，降低 `getaddrinfo ENOTFOUND api.cloudflare.com` 导致的获取失败问题。

## 发布说明

- 当前版本号：`v1.1.0`
- 发布方式：GitHub Release
- 本次发布不包含本地运行数据和证书数据文件
