# 更新记录

## v1.1.0 - 2026-03-19

### DNS 记录分页
- DNS 记录接口改为支持分页返回，前端按页读取 Cloudflare 解析记录。
- 记录列表新增上一页、下一页和页码输入，支持直接修改页数跳转。
- 页码区域改为显示总页数和总记录数，不再显示“第 1-10 条”这类摘要。

### 移动端与列表体验
- DNS 记录中间列表区域改为单独横向滚动，避免整页被撑出屏幕。
- 优化了手机端分页按钮、记录操作按钮、添加记录按钮和弹窗按钮的尺寸与排布。
- 加载状态改为显示在记录区域中央，并增加加载动画。
- 收紧了手机端“管理域名”头部和“批量修改”弹窗顶部留白，整体更紧凑。

### 页面结构与交互
- 调整域名详情头部结构，支持域名标题、添加记录按钮和批量操作按钮分别布局。
- 优化 PC 端分页区与操作按钮排序，减少错位问题。
- 移除部分多余图标，统一顶部和弹窗的视觉层级。

### Cloudflare API 连接修复
- 后端为 `api.cloudflare.com` 请求增加公共 DNS 兜底解析。
- 在本机系统 DNS 异常时，仍可通过公共解析器访问 Cloudflare API，减少 `getaddrinfo ENOTFOUND api.cloudflare.com` 问题。

### 说明
- 本次更新仅包含代码与文档调整。
- 本地运行数据与证书文件属于部署数据，不建议作为版本更新内容提交到仓库。
